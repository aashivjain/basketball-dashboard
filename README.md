## Features

- **Player Profiles** - 29 WNBA players with stats, game logs, performance trends
- **Team Pages** - Rosters, season stats, game predictions
- **Advanced Stats** - Shot charts, zone breakdowns, season comparisons
- **News Hub** - Latest WNBA articles
- **Performance Tracking** - Game-by-game stats, trend analysis
- **Mobile Friendly** - Works on phones and tablets

## Data

- **12,562 game records** from 2024-2026 WNBA season
- **29 player profiles** with full stats
- **News, team rosters, season stats, predictions**
- **100% data validated** - no corruption

---

## Stack

- **Backend:** Node.js + Express + SQLite
- **Frontend:** React + TypeScript + Vite
- **Data:** Python scripts for WNBA API integration
- **Security:** Rate limiting, SQL injection protection, CORS validation
